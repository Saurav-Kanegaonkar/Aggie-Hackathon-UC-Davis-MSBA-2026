# Ideas Folder

Drop working idea notes here so the team can share, compare, and refine approaches quickly during the hackathon.

Suggested file naming:
- `YYYY-MM-DD-<name>-<short-topic>.md`

Keep each file practical:
- problem framing
- why it matters to Fairlight
- candidate metrics or methods
- data needed
- risks / caveats
- next steps

This folder is intentionally lightweight. Fast notes are better than polished notes.
