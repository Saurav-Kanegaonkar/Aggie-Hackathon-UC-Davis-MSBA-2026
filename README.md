## Repo Structure

```
state/           — Project state files (index.json, events.jsonl)
state/schema/    — JSON schemas for validation
state/tasks/     — Per-task detail files
orchestrator/    — CLI orchestrator for task management
docs/            — Documentation and onboarding
history/         — Archived project snapshots
trial-calculator/ — Trial project (vanilla JS calculator)
```
